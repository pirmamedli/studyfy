# Studyfy

Адаптивный образовательный **PWA** для подготовки школьников к ЕГЭ — личный
кабинет ученика с заданиями, тестами, календарём, статистикой и геймификацией.
Построен с нуля на дизайн-системе **«Тёплый минимализм» v3**.

## Стек

- **React 18 + TypeScript + Vite**
- **React Router** — навигация (5 разделов + внутренние экраны)
- **vite-plugin-pwa** — установка на домашний экран, офлайн, service worker
- **Local-first + Supabase** — прогресс хранится в `localStorage` (офлайн-кэш) и
  синхронизируется с Supabase, когда заданы переменные окружения. Вход по
  email/паролю (Supabase Auth), данные — в таблице с RLS. Без переменных —
  чистый local-first режим с входом по коду доступа. Подробнее: [SUPABASE.md](SUPABASE.md).

## Ключевой принцип архитектуры

Единая связанная система. Все показатели **вычисляются из реальных действий**
пользователя, нигде не хранятся демо-значения:

```
предмет → тема → тест → попытка → ответы/ошибки → XP →
прогресс предмета → слабые темы → план и календарь
```

Единственный источник истины — `UserState` (`src/data/types.ts`).
Все экраны читают производные данные из `computeDerived()`
(`src/data/progress.ts`), поэтому после каждого завершённого теста
Главная, Прогресс, Материалы и Календарь обновляются автоматически.

## Структура

```
src/
  styles/         studify.css (дизайн-система) + app.css (каркас)
  data/
    types.ts      доменная модель (все сущности)
    content/      контент: subjects, topics, tests, materials
    ranks.ts      звания и пороги XP
    store.ts      local-first репозиторий + чистые редьюсеры
    progress.ts   производные показатели (XP, серия, точность, слабые темы)
  state/          AppProvider — контекст, персистентность, тема
  components/      AppShell (таббар + сайдбар), Ring
  screens/         Onboarding, Home, Tasks, Subject, TestRunner, TestResult,
                   Progress, Materials, Profile, Calendar, Mistakes
  lib/             иконки, иконки предметов, форматтеры, даты
design-source/     импортированная дизайн-система (референс): studify.css, экраны
```

## Запуск

> ⚠️ Node.js установлен в `~/.local/node` (нет в системном PATH).
> Добавьте его в PATH перед работой:
> `export PATH="$HOME/.local/node/bin:$PATH"`

```bash
npm install      # уже выполнено
npm run dev      # http://localhost:5173
npm run build    # прод-сборка + service worker
npm run preview  # предпросмотр сборки
npm run typecheck
```

Код доступа для входа (демо): `BETA-2026`.

## Реализовано

- 5 разделов нижнего меню + боковая панель на десктопе
- Обе темы (светлая/тёмная), автоопределение системной
- Онбординг/вход, дневной план, звания, серия, дни до ЕГЭ
- Прохождение теста (вперёд/назад, %, XP, зачёт, ошибки, избранное, заново)
- Прогресс: XP, лестница званий, точность, верно/ошибки, прогресс по предметам
- Материалы: библиотека, автоблок «Мои слабые темы», избранное, поиск
- Профиль: имя/ник/класс/дата ЕГЭ, выбор предметов, тема, цель на день,
  5 сценариев уведомлений, выход, удаление аккаунта
- Календарь активностей, дневник ошибок
